# EcoSphere AI 🌍

**Live Demo:** [https://prompt-wars-project-3.web.app](https://prompt-wars-project-3.web.app)

EcoSphere AI is a robust, production-ready sustainability platform featuring:
- **Carbon Intelligence Engine**: Mathematically verified emission calculations.
- **Explainable AI Coach**: Deeply integrated Gemini models exposing reasoning and data assumptions.
- **Eco Twin Simulator**: Dynamic "What-if" lifestyle scenario modeling.
- **Sustainability Planner**: Goal setting and tracking.
- **Community Leaderboard**: Hashed, anti-gaming integrity layers for global competition.

## Development

This project uses a monorepo structure with FastAPI (Python) and React (Vite).

### Quick Start
```bash
docker-compose up -d
```
Visit `http://localhost:5173` for the frontend and `http://localhost:8000/docs` for the Backend API.
