# purpose: Sustainability Methodology Documentation | enforces: Transparency, Quality-first

## 1. Sustainability Score Calculations
The base sustainability score is calculated on a 0-1000 scale.
- **Formula**: `max(0, 1000 - int(carbon_footprint_kg * 2))`
- **Bonuses**: Completed challenges add a static 50 points.

## 2. Benchmark Calculations
Percentiles are generated by comparing a user's CO2e against regional baselines.
- **Baselines**: NA (1500 kg), EU (800 kg), Global (1000 kg).

## 3. Carbon Calculation Methodology
Carbon calculations use static emission factors provided in `DATA_SOURCES.md`.

## 4. Leaderboard Ranking
Rankings are determined by total score, verified by an HMAC SHA-256 cryptographic hash to prevent score spoofing.
